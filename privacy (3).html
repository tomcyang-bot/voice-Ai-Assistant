隐私政策

我们非常重视您的隐私。本政策说明我们如何收集、使用和保护您的个人信息。

最后更新：2026年3月28日

核心承诺：您的数据存储在您自己的 Google 账号中，我们无法访问您的邮件原文和日历详情。我们只在您主动操作时临时处理必要的信息。

我们收集的信息

使用 CalVoice AI 时，我们会访问以下信息：

  • 您的 Google 账号基本信息（姓名、邮箱地址、头像），用于显示登录状态
  • 您的 Google Calendar 事件，用于显示和创建日程
  • 您的 Gmail 邮件元数据（发件人、主题），用于识别会议邀请
  • 您通过语音或文字输入的日程内容，用于 AI 分析和创建事件

我们如何使用您的信息

  • 分析您的语音/文字输入，提取日程信息
  • 在您的 Google Calendar 中创建事件
  • 扫描邮件主题，识别含会议信息的邮件
  • 生成邮件草稿（需要您确认后才发送）
  • 回答您关于日程安排的问题

我们不会做的事

  • 不会永久存储您的邮件内容或日历详情
  • 不会将您的数据出售或分享给第三方
  • 不会在未经您确认的情况下发送任何邮件
  • 不会读取与日程无关的邮件内容
  • 不会在您退出后保留任何个人数据

数据存储与安全

您的日历和邮件数据始终存储在您自己的 Google 账号中，我们通过 Google 官方 OAuth 2.0 授权访问，您可以随时在 Google 账号设置中撤销授权。

AI 分析过程中，您的输入内容会通过加密连接发送到 Anthropic Claude API 进行处理，处理完成后不会被永久保留。

第三方服务

本应用使用以下第三方服务：

  • Google OAuth / Gmail API / Calendar API — 用于账号登录和日程管理，受 Google 隐私政策约束
  • Anthropic Claude API — 用于 AI 分析，受 Anthropic 隐私政策约束
  • Vercel — 用于应用托管，受 Vercel 隐私政策约束

您的权利

  • 随时在 Google 账号设置中撤销本应用的访问权限
  • 要求删除与您账号相关的任何数据
  • 联系我们了解我们如何处理您的数据

Cookie 和本地存储

我们仅使用浏览器本地存储（localStorage）保存您的登录 Token，用于维持登录状态。退出登录后该 Token 会被立即清除。我们不使用任何追踪 Cookie。

政策更新

我们可能会不定期更新本隐私政策。重大变更时会在应用内通知您。继续使用本应用即表示您接受更新后的政策。
